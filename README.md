# Task 1: Web Scraping 

## Objective

The objective of this project was to collect data from publicly available websites and create a custom dataset suitable for analysis.

## Tools and Technologies

* Python
* Requests
* BeautifulSoup
* Pandas
* CSV

## Project Description

A web scraping solution was developed to extract book-related information from a public website. The extracted data was cleaned, structured, and stored as a CSV dataset for further analysis.

## Data Collected

The following information was extracted:

* Book Title
* Price
* Rating
* Availability Status

## Workflow

1. Identified a publicly accessible website.
2. Inspected HTML structure using browser developer tools.
3. Extracted required information using BeautifulSoup.
4. Cleaned and organized the collected data.
5. Stored the dataset in CSV format.
6. Performed basic exploratory analysis.

## Skills Demonstrated

* Web Scraping
* HTML Structure Analysis
* Data Collection
* Data Cleaning
* Dataset Creation
* Data Storage using CSV

## Output

* Custom Dataset (CSV)
* Python Scraping Script
* Basic Data Analysis

# Task 2: Data Visualization

## Objective

The objective of this project was to transform raw data into meaningful visualizations that communicate insights effectively and support data-driven decision-making.

## Tools and Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Description

A sales dataset was analyzed and visualized using multiple chart types to identify trends, patterns, and business insights.

## Visualizations Created

### 1. Sales by Category

A bar chart showing total sales generated by different product categories.

### 2. Profit by Category

A comparison of profit generated across categories.

### 3. Regional Sales Distribution

A pie chart illustrating sales contribution from different regions.

### 4. Monthly Sales Trend

A line chart used to identify sales patterns over time.

### 5. Sales vs Profit Analysis

A scatter plot highlighting the relationship between sales and profitability.

## Business Insights

* Certain categories generated significantly higher sales than others.
* Profitability varied across product categories.
* Regional performance revealed key revenue-generating areas.
* Seasonal trends were observed in monthly sales performance.
* High sales volumes did not always correspond to high profits.

## Skills Demonstrated

* Data Visualization
* Exploratory Data Analysis
* Business Intelligence
* Data Storytelling
* Insight Generation
* Dashboard Thinking
